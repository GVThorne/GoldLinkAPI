To install and use follow the steps below:

1. Unzip the folder into the root directory of your PHP enabled web server
2. Open the file GVGoldLinkNTLM.php file for editing
3. Change the variables $user, $password and $GVAddress for your installation

Explanations for each file are below:
* Example1.php - Outputs basic Version and Licencing information
* Example2.php - Outputs the summary's of all Accounts in your Gold-Vision
* Example2.php?searchterm=[SEARCHTERM] - Outputs the summary's of all Accounts in your Gold-Vision that match [SEARCHTERM]

N.B. These files exist only to provide an example of how to interact with the Gold-Link API using PHP; they are not suitable for production deployment as is.
